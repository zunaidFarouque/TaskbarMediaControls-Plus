# TaskbarMediaControls Windows 11

**TaskbarMediaControls** is a lightweight Windows tray application that lets you control media playback directly from the system tray. It provides **Previous, Play/Pause, and Next** buttons as individual tray icons for quick access to media controls without opening your player.  

I originally got the idea from a GitHub repository that I can no longer find, but I liked the concept so much that I decided to recreate it.

### Features
- Control media playback (Previous, Play/Pause, Next) with a single click  
- Toggle Play/Pause with visual icon update  
- Optional launch on Windows startup  
- Customizable via tray context menu  
- Lightweight and always-on-top tray presence  

### Limitations
- ~~**Light mode** may not display correctly, as I did not bother creating new icons for it~~
- **VLC** is currently not supported. I am working on supporting it though!
- Additional features may be added if there’s enough interest from users.  

### Installation
1. [Download the latest installer](https://github.com/2latemc/TaskbarMediaControls/releases/latest)
2. Run the app once
3. Drag and Drop the Tray Icons from the "show more" menu if collapsed
<img width="255" height="157" alt="taskbar" src="https://github.com/user-attachments/assets/3cf2f980-81c3-4093-9be5-dd99d55e6d4a" />

6. Use the tray menu to enable/disable launch on startup or exit the app.
